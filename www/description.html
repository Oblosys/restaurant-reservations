Description

Web-app for reservations

Features

Features Browsing/editing existing reservations, one interface for clients who make new reservation. Dynamic interface, cannot make reservations for full time slots Real-time updates Works on mobile Open second window in browser or on mobile device. Immediate updates. Keyboard support tab between calendar, day view, and reservation view. Arrow keys to navigate between days and reservations. Delete to delete reservation.

Technical details

Shared modules between client & server JQuery, Backbone data model, Underscore Server implemented with Node.js and express Push html with socket.io
